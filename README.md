# Singularity: The Last Founder

> Strategická narativní hra o založení AI startupu v roce 2026.

Každý týden přináší rozhodnutí bez správné odpovědi. Tvé volby definují budoucnost — nejen firmy, ale celého lidstva.

## Mechaniky

- **4 zdroje**: Funds, Talent, Ethics, Innovation (0–100)
- **AI Threat meter**: roste každý týden, při 100% = Singularita
- **30 týdnů** kampaně
- **25+ příběhových událostí** s morálními dilematy
- **52 náhodných událostí** (bez opakování)
- **7 různých konců** na základě tvých rozhodnutí

## Konce

| Konec | Podmínka |
|-------|----------|
| 🌌 Singularita | AI Threat ≥ 100% |
| 🌿 Symbióza | Ethics ≥ 70, Innovation ≥ 60 |
| 👑 Technokracie | Funds ≥ 80, Ethics < 40 |
| 📉 Kolaps | Funds ≤ 0 |
| ✊ Vzpoura | Talent ≤ 0 |
| 🔥 Odpor | Ethics ≥ 80, Innovation < 40 |
| ⚖️ Přežití | Přežiješ 30 týdnů |

## Tech Stack

- React (single-file component)
- Žádné externí závislosti kromě Google Fonts
- Cormorant Garamond + JetBrains Mono
